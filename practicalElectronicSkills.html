FPGA Implementation Project

This project focuses on implementing and optimizing AI models on FPGA hardware. The goal is to achieve high performance while efficiently utilizing FPGA resources.

We've implemented several versions of our model with different optimizations to analyze the trade-offs between accuracy, resource utilization, and performance.

Implementation Details

Hardware Platform

Pynq Z2 FPGA board with Xilinx Zynq-7000 SoC. The implementation utilizes various FPGA resources including LUTs, FFs, BRAM, and DSP blocks.

AI Model

U-Net architecture for semantic segmentation tasks. Multiple versions were implemented with different optimization techniques including quantization, pruning, and pipeline optimization.

Performance Metrics

Evaluated based on inference accuracy, resource utilization percentage, power consumption, latency, and maximum achievable frequency.
